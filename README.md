## 🚀 使用指南

### 构建

本项目现在只提供 Rust 构建方式：

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\build-rust.ps1
```

构建产物：
- `target\release\NapCatWinBootMain.exe`
- `target\release\NapCatWinBootHook.dll`

启动器会优先使用同目录下的 `QQ.exe`，找不到时会从注册表读取 QQ 安装路径。

项目结构：
- `crates/launcher`：启动器，负责定位 QQ 并注入 DLL
- `crates/hook`：一次性内存补丁 DLL，完成后自动卸载
- `scripts/build-rust.ps1`：Windows Rust 构建脚本

### 安装要求
- Windows 操作系统
- 已安装的 QQ 客户端

### 安装步骤

#### 步骤 1：下载文件
前往 [Releases](../../releases) 页面下载最新版本的压缩包

#### 步骤 2：解压安装
1. 解压下载的压缩包
2. 将解压出的 **两个文件** 复制到 QQ 安装目录
   - 默认路径通常为：`C:\Program Files\Tencent\QQ\` 
   - 或在 QQ 快捷方式上右键 → 属性 → 打开文件所在位置
3. 如果实在找不到你就随便放到一个位置, 重新会通过读取注册表判断QQ路径

#### 步骤 3：创建快捷方式
为 `NapCatWinBootMain.exe` 创建桌面快捷方式：
1. 右键点击 `NapCatWinBootMain.exe`
2. 选择 "发送到" → "桌面快捷方式"

#### 步骤 4：启动程序
1. 双击桌面上的快捷方式
2. 等待程序加载完成
3. 正常登录你的QQ就行了

### 使用说明
- 启动成功后，程序会自动注入防撤回功能
- 正常使用 QQ，防撤回功能已在后台生效
- 无需额外操作，群聊和私聊撤回消息都会被拦截并保留原消息
- 自己发起撤回时，对外仍正常撤回，本地保留原消息可见

## ✨ 功能特性

### 优点
- 🔄 **持久化拦截**：无消息数量约束，持续有效
- 📁 **资源保护**：图片/文件下载不会在撤回后失败  
- 🛡️ **低风控风险**：检测特征极少，启动后进程分离并取消DLL注入
- 👤 **自撤回保留**：自己撤回的消息对外撤回成功，本机仍可查看

### 局限性
- 🔧 **版本兼容性**：适配版本较少, 但仍可尝试不同版本
- ⚠️ **稳定性**：可能出现程序崩溃情况
- 🔍 **检测风险**：虽然触发较少，但仍存在检测可能性
- 👥 **功能范围**：目前实现群聊和私聊防撤回
